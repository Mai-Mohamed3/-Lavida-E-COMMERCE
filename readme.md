# 🛒 E-Commerce Amado

## 📌 Description  
E-Commerce Amado is a modern and stylish online shopping website that allows users to browse products, add items to their cart, and complete purchases seamlessly. The project is designed with a user-friendly interface and smooth interactions.

---

## 🚀 Technologies Used  
- **Frontend:** HTML, CSS, JavaScript
- **Libraries & Frameworks:** Font Awesome
- **Tools:** Git, VS Code

---

## 📂 Installation  

1. **Clone the repository:**  
    ```bash
    git clone https://github.com/username/e-commerce-amado.git
    ```
    ```bash
    cd e-commerce-amado
    ```

2. **Open the project in a browser:**  
   Simply open `index.html` in your browser to view the website.

---

## 🛠 Key Features  
✅ Mobile-friendly responsive design  
✅ Side navigation menu for easy browsing  
✅ Categories with hover effects  
✅ Shopping cart functionality  
✅ Subscription form for discounts  
✅ Scroll-to-top button for better navigation
